  11. docker restart container_id_or_name -

  13. docker logout - разлогинется
  14. docker run -it --name My_project openjdk - запуск проекта по имени в интерактивном режиме
  15. ctrl + d - выход из контейнера

Скачивание и запуск образов

  1. Скачать Docker desktop
  2. Зарегистрироваться на docker hub https://hub.docker.com
  3. Перейти на страницу с образами https://hub.docker.com/explore

Команды для работы с образами

  1. docker pull (имя образа) скачивает образ по имени
  2. docker pull (имя образа):(номер версии или last) скачивает образ по имени
  3. docker run (имя образа) запускает образ и создает контейнер с рандомным именем
  4. docker run -it --name(имя контейнера) (имя образа) запускает образ в интерактивном режиме и создает контейнер с именем (имя контейнера)
  5. ctrl + d - Выход из контейнера
