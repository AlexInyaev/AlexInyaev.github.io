1. docker loginАвторизуемся внутри терминала
  2. docker build -t myLogin/nameImage ./php -t myLogin/nameImage - мой логин и имя проекта которое сам назову ./php- папка где лежит dockerfile
  3. docker images после сборки можем проверит наличие образа но не обязательно
  4. docker push myLogin/nameImage:latest :latest - номер версии можно не писать
  5. https://hub.docker.com/repositories/... в docker hub появится наш новый проект
  6. Так же можно скачать образ для этого открываем его в docker hub и копируем там команду типа этой - docker pull alexinpnz/testphp
  7. При загрузки образов в докер все описание (номера портов и.т.д. пишем в overview на сайте)
  8. docker run -p 3001:80 --name myapp alexinpnz/testphpЗапускаем скаченный проект
