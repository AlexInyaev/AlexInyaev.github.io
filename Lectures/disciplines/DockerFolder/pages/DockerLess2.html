1. Создаем DockerFile для программы на java
      поля DockerFile :
    • FROM openjdk Образ который будем загружать
    • COPY . /java Первая точка от куда копируем файлы (если точка то эта папка) /java - куда копировать
    • WORKDIR /java Рабочая директория
    • EXPOSE 8001 Указывает порт
    • RUN javac Main.javaЗапускается один раз при создание образа
    • CMD ["java","Main"]Запускается каждый раз при запуску контейнера. ["java","Main"] - Это такой синтаксис Если писать в терминале то java Main
  2. Создаем файл в корне проекта Main.java с кодом 
            
          class Main {
            public static void main(String[] args) {
                System.out.println("Hello World!");
            }
        }
  3. docker build . собирает проект, точка в конце обязательна
  4. docker images смотрим наш образ
  5. docker run -p 3001:8001 (id - образа)Запуск проекта, -p 3001:8001 - соединяет порты 3001 на компе
  6. docker image rm (id - образа) удаляет образ
  7. Структура проекта Project > Dockerfile Maim.java
