Команды

Базовые команды Windows

  • dir - показывает директории
  • cd - смена директории поддерживается относительный и абсолютный путь
  • cd ./шаг вперед
  • cd ..шаг назад
  • mkdir name-folder - создание папки
  • echo hi > index.txt создает файл и передает в него hi
  • cat index.txt показывает содержимое файла
  • del .\index.txt удаляет файл
  • copy .\folder1 .\folder2 копирование папок
  • move .\folder1 .\folder2 перемещение папок

Работа с Git - 3

  • code . - открыть vs code
  • git init - инициализирует репозиторий в ту папку в которой исполняется
  • git status - Статус гита
  • git config user.name - проверяет или устанавливает имя пользователя в git
  • git config user.email - проверяет или устанавливает email пользователя в git
  • git config -global user.name - проверяет или устанавливает имя пользователя в git глобально
  • git config -global user.email - проверяет или устанавливает email пользователя в git глобально
  • git config list - список всех конфигураций
  • git add file-name - добавление файла в отслеживание гитом
  • git add . - добавление всех файлов в отслеживание гитом
  • git rm -cached file-name - убирает файл из отслеживания
  • git commit -m "описание коммита" - делаем коммит
  • git log - информация о каммитах
  • git checkout №коммита - переход к нужному коимиту
  • git checkout master - возврат в ветку мастер
