Добавление раздела

  1. В папке disciplines дублируем папку TemplateFolder меняя название папки на свое и таг-же меняем название файлов в этой папке
  2. В файле index.html в раздел Пункты меню второго уровня (отмечен комментарием) находим дублируем строку снизу : 
        <li id="TemplateId" class="hoverEffect">Template</li>
    и меняем названия атрибута id на свое : TemplateId - будем использовать для навешивания события onClick
  3. B раздел Пункты меню третьего уровня находим и дублируем код ниже и все Template заменяем на свои значения:
                <!-- Название предмета -->

<ol id="TemplateMenu"> //здесь берем атрибут ol для переменной blockMenu
  //в  функцию makeMenu
    <!-- Id элементов li  должны совпадать с названиями файлов в папке 
    TemplateFolder/pages на этом строится логика отрисовки страниц -->
    <li id="templateLess1" class="templateLess hoverPointer">Lesson1 Template</li> 
    // class="templateLess" используется в функции makeMenu 
    <li id="templateLess2" class="templateLess hoverPointer">Lesson2</li>
</ol>
  4. В файле index.js в разделе "Добавление Раздела и обработчиков на пункты меню" (выделено комментариями) дублируем код :
    document.querySelector("#TemplateId").addEventListener("click", () => {
  parametersDirectory = makeMenu(event, templateMenu, "templateLess", "TemplateFolder");
  addEvents();
});

    Описание кода

    document.querySelector("#TemplateId").addEventListener("click", () => {
parametersDirectory = makeMenu(event, blockMenu, classDirectory, nameDirectory);
addEvents();
});

    Далее нам нужно изменить параметры функции makeMenu

    Чтобы получить blockMenu нужно в файле variables найти продублировать строку : const templateMenu = document.querySelector("#TemplateMenu"); и заменить в ней templateMenu на свое значение

    classDirectory : это класс который навешен на все пункты меню

    nameDirectory : это папка TemplateFolder (название папки проекта)

    Находим функцию returnToStartMenu() в ней находим и дублируем строку templateMenu.style.display = "none"; в которой меняем templateMenu на свое

    CSS для id тега ol (TemplateMenu) прописать display:none

    1. event - объект события , остается без изменения
    2. blockMenu - полный список меню третьего уровня полученный по Id пример : 
      const englandMenu = document.querySelector("#EnglandMenu");
    3. classDirectory - этот класс должен быть прописан в каждом li пункта меню
    4. nameDirectory - название папки директории в которой хранятся файлы страниц используется для построения пути
