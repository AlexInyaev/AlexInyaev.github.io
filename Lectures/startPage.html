Добавление раздела

  1. В папке disciplines добавляем папку с новой дисциплиной с вложенной папкой pages в которой будут лежать страницы уроков/разделов
  2. В файле index.html в раздел Пункты меню второго уровня (отмечен комментарием) добавляем в список пункт меню: 
        <li id="Eng" class="hoverEffect">English</li>
  3. B раздел Пункты меню третьего уровня добавляем меню для нового раздела это выглядит так:
                <!-- Название предмета -->

    <ol id="EnglandMenu"> //здесь берем атрибут ol для переменной blockMenu в  функцию makeMenu
            <!-- Id должен совпадать с названием файла -->
            <li id="engLess1" class="engLess hoverPointer">Lesson1 Eng</li> 
            // class="engLess" используется в функции makeMenu 
            <li id="engLess2" class="engLess hoverPointer">Lesson2</li>
    </ol>
  4. В файле index.js в разделе "Добавление Раздела и обработчиков на пункты меню" (выделено комментариями) добавляем код :
    document.querySelector("#Eng").addEventListener("click", () => {
parametersDirectory = makeMenu(event, blockMenu, classDirectory, nameDirectory);
addEvents();
});

    Далее нам нужно изменить только параметры функции makeMenu

    1. event - объект события , остается без изменения
    2. blockMenu - полный список меню третьего уровня полученный по Id пример : 
      const englandMenu = document.querySelector("#EnglandMenu");
    3. classDirectory - этот класс должен быть прописан в каждом li пункта меню
    4. nameDirectory - название папки директории в которой хранятся файлы страниц используется для построения пути
