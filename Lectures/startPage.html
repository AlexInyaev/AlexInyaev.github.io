📚 Добавление нового раздела в проект

Следуйте этой инструкции, чтобы добавить новый раздел с уроками.

1 Создать папки и файлы

В папке disciplines/ создайте новую папку с названием вашего раздела (например, MyNew).

Внутри создайте папку pages/ и добавьте файлы:

disciplines/
└── MyNew/                    <-- название вашего раздела
    └── pages/
        ├── myNewLessMarkup.html   <-- стартовая страница раздела (обязательно!)
        ├── myNewLess1.html        <-- урок 1
        ├── myNewLess2.html        <-- урок 2
        └── ...                    <-- добавляйте сколько нужно
💡 Важно: Файл myNewLessMarkup.html ОБЯЗАТЕЛЕН! Он загружается первым при выборе раздела.

2 Добавить пункт в главное меню (index.html)

Откройте файл index.html и найдите секцию ПУНКТЫ МЕНЮ ВТОРОГО УРОВНЯ (ГЛАВНОЕ МЕНЮ).

Добавьте строку:

<li id="MyNew" class="hoverEffect">MyNew</li>
💡 Пояснение:
• id="MyNew" — уникальный идентификатор (будет использован в JavaScript)
• class="hoverEffect" — не меняйте (отвечает за стили при наведении)
• MyNew — текст, который увидят пользователи в меню

3 Добавить меню уроков (index.html)

В том же файле index.html найдите секцию ПУНКТЫ МЕНЮ ТРЕТЬЕГО УРОВНЯ (УРОКИ).

Добавьте блок:

<ol id="MyNewMenu">
  <li id="myNewLess1" class="myNewLess hoverPointer">Урок 1</li>
  <li id="myNewLess2" class="myNewLess hoverPointer">Урок 2</li>
  <li id="myNewLess3" class="myNewLess hoverPointer">Урок 3</li>
</ol>
💡 Пояснение:
• id="MyNewMenu" — идентификатор блока (нужен для JavaScript)
• id="myNewLess1" — должен СОВПАДАТЬ с именем файла урока (без .html)
• class="myNewLess" — класс для группировки уроков (должен быть одинаковым у всех уроков раздела)
• Текст между тегами — то, что увидят пользователи

4 Добавить переменную в variables.js

Откройте файл scripts/variables.js и найдите секцию МЕНЮ ТРЕТЬЕГО УРОВНЯ.

Добавьте строку:

const MyNewMenu = document.querySelector("#MyNewMenu");

5 Добавить обработчик в index.js

Откройте файл scripts/index.js и найдите секцию ОБРАБОТЧИКИ ГЛАВНОГО МЕНЮ.

Добавьте код:

document.querySelector("#MyNew").addEventListener("click", (event) => {
  if (!checkElement(MyNewMenu, 'MyNewMenu')) return;
  parametersDirectory = makeMenu(event, MyNewMenu, "myNewLess", "MyNew");
  if (parametersDirectory) addEvents();
});
💡 Пояснение параметров makeMenu:
• event — объект события (не меняйте)
• MyNewMenu — переменная из variables.js
• "myNewLess" — класс уроков (из шага 3)
• "MyNew" — название папки в disciplines/

6 Добавить в функцию returnToStartMenu (index.js)

Найдите функцию returnToStartMenu() и массив allMenus.

Добавьте вашу переменную:

const allMenus = [
  englandMenu, russianMenu, phpMenu, GolangMenu,
  DockerMenu, GitCourseMenu, templateMenu, ServerAppMenu,
  MyNewMenu   <-- добавить сюда
];

7 Добавить стили в style.css

Откройте файл styles/style.css и найдите секцию МЕНЮ ТРЕТЬЕГО УРОВНЯ.

Добавьте правило:

#MyNewMenu {
  display: none;
}

8 Создать стартовую страницу раздела

Создайте файл disciplines/MyNew/pages/myNewLessMarkup.html.

Минимальное содержимое:

<!doctype html>
<html lang="ru" data-theme="dark">
<head>
  <meta charset="UTF-8">
  <title>Мой новый раздел</title>
  <link rel="stylesheet" href="../../../libs/pico-main/css/pico.css">
  <link rel="stylesheet" href="../../../libs/highlight/styles/tomorrow-night-bright.css">
  <link rel="stylesheet" href="../../../styles/styleForPages.css">
</head>
<body>
  <div class="container">
    <h1>Добро пожаловать в раздел "MyNew"</h1>
    <p>Выберите урок из меню слева.</p>
  </div>
  <script src="../../../libs/highlight/highlight.pack.js"></script>
  <script>hljs.initHighlightingOnLoad();</script>
</body>
</html>

9 Создать файлы уроков

Создайте файлы уроков в папке disciplines/MyNew/pages/:

  • myNewLess1.html — первый урок
  • myNewLess2.html — второй урок
  • и т.д. (сколько нужно)
💡 Важно: Имена файлов должны СОВПАДАТЬ с id пунктов меню из шага 3!

10 Проверка

После выполнения всех шагов:

  1. Откройте консоль браузера (F12)
  2. Обновите страницу (F5)
  3. Кликните на ваш новый раздел в меню
  4. Проверьте, что загрузилась страница myNewLessMarkup.html
  5. Кликните на урок — должна загрузиться соответствующая страница
⚠️ Если что-то не работает:
• Проверьте консоль на наличие ошибок 404 (файл не найден)
• Убедитесь, что все имена совпадают (регистр важен!)
• Проверьте, что вы добавили все пункты во все файлы

📋 Шпаргалка: что куда добавлять

1. disciplines/MyNew/pages/myNewLessMarkup.html     ← стартовая страница
2. disciplines/MyNew/pages/myNewLess1.html         ← урок 1
3. index.html → mainMenu → <li id="MyNew"...>    ← пункт главного меню
4. index.html → блоки уроков → <ol id="MyNewMenu"> ← меню уроков
5. variables.js → const MyNewMenu = ...            ← переменная
6. index.js → обработчик для #MyNew                ← обработчик клика
7. index.js → returnToStartMenu → добавить в массив ← скрытие меню
8. style.css → #MyNewMenu { display: none; }       ← скрыть по умолчанию
✅ Готово! Теперь вы можете добавлять неограниченное количество новых разделов, следуя этой инструкции.
