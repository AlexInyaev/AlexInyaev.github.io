Работа в приложении

  1. Выбираем нужную нам категорию
  2. В карточке появится слово вопрос на которое пишем ответ ниже и нажимаем enter или кнопку ответ. Повторное нажатие переключает слово
  3. Кнопка очистить и клавиша Ctrl + x очищают поле
  4. Если 3 раза дать неправильный ответ то в низу высветится правильный ответ
  5. Для получения быстрого ответа нужно нажать на Быстрый отв. И затем для переключения на следующие слово нажать на Ответ

Добавление директории слов

  1. В dictionaries.js ищем массив pattern и дублируем сам "pattern" - меняем на свое значение
  2. В index.html ищем select и дублируем опцию pattern сам "pattern" - меняем на свое значение
  3. В selectDictionary.js ищем switch и дублируем кейс pattern сам "pattern" - меняем на свое значение
