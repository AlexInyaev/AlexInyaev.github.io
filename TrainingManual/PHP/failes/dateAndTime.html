Дата и время

Функциональный способ

        
            date_default_timezone_set('Europe/Moscow'); //устанавливает часовой пояс по умолчанию для всех функций даты/времени в скрипте.
            echo '<br>';
            echo date('h i s'); //выводит время дополнительно читать в мануале
            echo '<br>';
            echo date('Время сейчас - h:i:s'); //можно также добавлять слова в вывод символы между выводами
            echo '<br>';
            echo time(); //возвращает время в миллисекундах с 01.01.1970
            echo '<br>';
            // с помощью time(); можно устанавливать время например выводим минус час
            echo date('Время сейчас - h:i:s',time() - 3600);
            echo '<hr>';

Объектный способ

        
            $date_now = new DateTime(); //создаем объект
            $date_now->modify('+1 hour'); // можно модифицировать на любое время 
            echo $date_now->format('Время сейчас - h:i:s'); // вывод время в формате
