Подключение файлов

        
            require 'func.php';         // при ошибке скрипт остановится. 'func.php' -имя подключаемого файла
            include 'func.php';         // при ошибке скрипт продолжит работать
            
            require_once 'func.php';    // если 2 раза подключить скрипт таким способом ошибки не будет
            include_once 'func.php';    // если 2 раза подключить скрипт таким способом ошибки не будет

При подключении файла также становятся видны все данные из подключаемого файла
