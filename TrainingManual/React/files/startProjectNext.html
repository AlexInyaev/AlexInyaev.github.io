Start Project Next

  1. Устанавливаем node JS
  2. Устанавливаем github
  3. Создаем папку для программы и прописываем путь в консоли/гитбаше или открываем эту попку в vsCode и используем встроенный терминал для команд
  4. Инициализирует проект команда: npm init -y после появится файл package.json c default настройками
  5. Устанавливаем nextJS команда: npm install next@latest react@latest react-dom@latest появится папка node modules
  6. Создаем папку pages в этой папке создаем файл index.jsx в файл прописываем функцию export default function app(){return jsx}
  7. Запускаем проект команда: npx next dev копируем адрес локохоста и вставляем в браузер и переходим
