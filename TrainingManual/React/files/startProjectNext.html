Start Project Next

  1. Устанавливаем node JS
  2. Устанавливаем github
  3. Создаем папку для программы и прописываем путь в консоли/гитбаше или открываем эту попку в vsCode и используем встроенный терминал для команд
  4. Инициализирует проект команда: npm init -y после появится файл package.json c default настройками
  5. Устанавливаем nextJS команда: npm install next@latest react@latest react-dom@latest появится папка node modules
  6. Создаем папку pages в этой папке создаем файл index.jsx в файл прописываем функцию export default function app(){return jsx}
  7. Запускаем проект команда: npx next dev копируем адрес локохоста и вставляем в браузер и переходим 
        
           # Ваш Код

Tailwind css

  1. Устанавливаем Зависимости команда npm install -D tailwindcss postcss autoprefixer
  2. Генерируем конфигурацию с помощью Tailwind npx tailwind css init -p появится файл tailwind.config.js открываем его и заменяем массив content на 
                    
                    content: [
                                "./app/**/*.{js,ts,jsx,tsx,mdx}",
                                "./pages/**/*.{js,ts,jsx,tsx,mdx}",
                                "./components/**/*.{js,ts,jsx,tsx,mdx}",
                            
                                // Or if using `src` directory:
                                "./src/**/*.{js,ts,jsx,tsx,mdx}",
                            ],
  3. в корне создаем папку styles добавляем в нее файл global.css и прописываем туда 
                    
                    @tailwind base;
                    @tailwind components;
                    @tailwind utilities;
  4. В папке pages Создаем файл _App.jsx (который является глобальным для проекта) и подключаем туда стили 
                    

                    import "../styles/global.css";

                    export default function App({ Component, pageProps }) {
                        return <Component {...pageProps} />;
                    }
  5. Проверяем в файлу global.css пишем фон для боди для проверки
  6. Для удобства можно установить плагин Tailwind CSS IntelliSense - При нажатии ctrl+пробел показывает окно с классами
