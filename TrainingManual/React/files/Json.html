Можно также создать файл json при помощи команды консоли

Первая команда : npm init - после этой команды нужно несколько раз подтвердить данные или изменить их

Вторая команда : npm init -y - все подтвердится автоматически

Описание версий библиотек

semver - Семантическое Версионирование Подробное описание

        
            пример названий индексов версии описание выше
            "react-dom": "^18.2.0",
            ^- - Наличие означает что версия может изменится чтобы сохранить текущею версию кружку уберем
            18 - МАЖОРНАЯ - может обновление может поломать сайт надо читать документацию что поменялось
            2 - МИНОРНУЮ - добавлена функциональность нечего не ломается в сайте и приносит новые фитчи в библиотеке
            0 - ПАТЧ- не чего не добавляет фиксит баги

PS мелкие библиотеки могут не придерживаися этих правил

npm npx установка библиотек

npx - скачивает выполняет код библиотеки чтобы получить нужную структуру а исходники потом удаляются

npm - скачивает и устанавливает библиотеку 

      
npm install name library - установится последняя версия библиотеки
npm uninstall name library - удаление библиотеки
короткая запись 
npm i name library - установится последняя версия библиотеки
npm un name library - удаление библиотеки

npm i name library@number version - установка заданной версии библиотеки
npm i name library@number version -E - установка заданной версии библиотеки которая не будет обновятся (версия будет без ^)

npm un name-library name-library через пробел можно сразу указать две библиотеки для установки
