Расширения и настройки VScode

  1. Эммет и автозаполнение работает из коробки
  2. Авто сохранение - file -> Preference -> Settings -> пишем в поисковике auto save -> выбираем on focus change
  3. live server : нажимаем расширения -> в поисковике пишем live Server -> install ->перезагружаем
  4. Toggle Format - Выравнивание кода : расширения -> ищем и устанавливаем Toggle Format on Save затем Preference -> Settings -> ищем Format On Save и ставим галку
  5. Prettier - Выравнивание кода. заходим на сайт и там берем команды:
    1. установка команда консоли: npm install --save-dev --save-exact prettier
    2. Создаем файл в корне проекта .prettierrc.json и в файл кладем пустой объект {}
    3. Создаем файл в корне проекта .prettierignore и в файл прописываем то что притер будет игнорировать .next - ее не нужно форматировать
    4. Затем выполняем команду npx prettier . --write
    5. Устанавливаем плагин Prettier code formatter
    6. Зажимаем ctrl + shift + p вводим format document
    7. Preference -> Settings -> ищем Format On Save и ставим галку для авто форматирования при сохранение

  6. ESLint - Статический анализатор исходного кода

    1. Установка команда консоли:npx next lint (выбираем строгий режим)
    2. Для проверки набираем такую-же команду консоли npx next lint
    3. Или Подключение плагина который будет показывать ошибки автоматически
      • Вводим в поиске расширений eslint и устанавливаем
      • После весь текст с ошибками будет подчеркнут и если навести курсор то высветится ссылка перейдя по которой можно почитать про ошибку
  7. emmet для react: file -> Preference -> Settings -> пишем в поисковике emmet-> ищем Emmet: Include Languages-> жмем add item
    где key пишем javascript где value пишем javascriptreact
  8. Bracket pair colorizer - подсветка скобок
  9. Auto rename tag - при изменение тега второй тег меняется автоматом
  10. JavaScript (ES6) code snippets- сниппеты js
  11. ES7+ React/Redux/React-Native snippets -сниппеты реакт
  12. Quokka.js -используется для отладки кода
  13. Russian - Code Spell Checker - проверка ошибок устанавливаем расширение затем жмем F1 и вводим Enable Russian Spell Checker Dictionary и Enable Russian Spell Checker Dictionary in Workspace
  14. CSS Peek - Позволяет находить выделенный класс в html в css: нужно выделить нужный класс и нажать пкм затем go to definition или f12
  15. Path autocomplete помогает прописать путь к файлам
  16. Lorem ipsum Добавление слов в теги пишем lorem5 = добавить 5 слов
  17. https://wordsonline.ru/orthography/check.html -- сайт для проверки слов
  18. https://www.reverso.net/проверка-орфографии/английский/ --
