Сниппеты

Настройка сниппетов

Мы можем использовать расширение Java code snippets которое имеет описание своих сниппетов

Настройка сниппетов в VS code без расширений

Видео с описанием настройки
  1. Кликаем на шестеренку в нижнем правым углу и выбираем snippets
  2. Выбираем язык для которого пишем сниппет
  3. Откроется файл json там будет пример. Подробное описание примера я сделаю ниже

Пример целиком

        
            {
                // Place your snippets for css here. Each snippet is defined under a snippet name and has a prefix, body and 
                // description. The prefix is what is used to trigger the snippet and the body will be expanded and inserted. Possible variables are:
                // $1, $2 for tab stops, $0 for the final cursor position, and ${1:label}, ${2:another} for placeholders. Placeholders with the 
                // same ids are connected.
                // Example:
                // "Print to console": {
                // 	"prefix": "log",
                // 	"body": [
                // 		"console.log('$1');",
                // 		"$2"
                // 	],
                // 	"description": "Log output to console"
                // }
            }

Моё описание примера

        
            {
                 "Print to console": {
                 	"prefix": "log",  --- это команда которую пишем и нажимаем enter
                 	"body": [
                 		"console.log('$1');",  ---- код который выведется
                 		"$2"
                 	],
                 	"description": "Log output to console" --- название сниппета не обязательно писать
                 }
            }

$1,$2,$3 итд - это точки где будет курсор изначально он появится где $1 затем при нажатие tab он сместится на $2

Если написать в одном сниппете несколько $1 $1 или $2 $2 $2 то курсор появится одновременно в тех местах где они написаны
