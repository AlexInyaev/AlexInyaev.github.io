Таблицы

        
        <table>
            <tr>
                <th>ячейка - заголовка</th>
                <td>ячейка2</td>
                <td>ячейка3</td>
            </tr>
        </table>

        colspan=""  - объединяет колонки
        rowspan=""  - объединяет ряды

        стили:
        border: 1px solid grey;    - 
        border-collapse: collapse; - Устанавливает, как отображать границы вокруг ячеек таблицы.

        tr:nth-child(even){bacgraund:silver;} - Окрашивает четные ряда табл.
