Интерфейс Трейт Абстрактный класс

Класс мы не можем наследовать от одного класса, с этим справляются интерфейсы 

        
            <?php
            interface Human{             //создание интерфейса 
                public function talk();
                public function walk();
            }
            interface Mutant{              //создание интерфейса 
                public function fly(); 
            }
        
            class Person implements Human, Mutant{       //наследование от двух интерфейсов. При наследование от интерфейса в классе должны  быть реализованы все методы интерфейса
                function talk(){
                    echo 'say' , '<br>';
                }
        
                function walk(){
                    echo 'go' , '<br>';
                }
        
                function fly(){
                    echo 'go to fly' , '<br>';
                }
            }
        
            $bob = new Person; //создаем объект на основе класса 
        
            $bob -> talk();
            $bob -> walk();
            $bob -> fly();
        
            // ****************** также интерфейсы могут наследоваться 
            interface Test extends Mutant{
        
            }
            
            class Test2 implements Mutant{
                function fly(){
                    echo 'test2';
                }
            }
        
            $bob2 = new Test2;
            $bob2 -> fly();
            ?>

В трейте мы можем прописать функции и затем их использовать в множестве методов 

        
            <?php  
            trait PrintSome{            //создаем трейт 
                public function talk(){
                    echo 'Привет всем <br>';
                }
                public function SeyBay(){
                    echo 'Пока всем <br>';
                }
            }
        
            trait SecondTrait{
                public function secondFunction(){
                    echo 'secondFunction';
                }
            }
        
            class TestTr{
                // use PrintSome,SecondTrait;      //добавляем трейт в класс 
                use PrintSome,SecondTrait;      //добавляем второй трейт в класс 
            }
        
            $obj = new TestTr;
            $obj -> talk(); //используем функции трейта в объекте
            $obj -> SeyBay();
            $obj -> secondFunction();  //вызываем метод из второго трейта
            ?>

Абстрактные классы это те классы на основе которых нельзя создать объекты

В абстрактных классах могут быть абстрактные методы которые должны быть реализованы в классах наследниках 

        
            <?php
    abstract class Car { //создаем абстрактный класс
        protected $speed;
        protected $color;

        abstract protected function showInfo(); //создаем абстрактный метод 
    }

    class BMW extends Car {  //наследуемся от абстрактного класса

        function __construct($speed){
            $this -> speed = $speed;
        }
        public function showInfo(){                 // Обязательно реализуем абстрактный метод
            echo  'Скорость автомобиля: ' . $this -> speed; 
        }
    }
    
    $m3 = new BMW(240);
    $m3 -> showInfo();
    ?>
