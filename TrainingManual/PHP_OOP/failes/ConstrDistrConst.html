Конструкторы , Деструкторы , Константы

Конструктор - это обычная функция которая позволяет нам при создание объекта присвоить значения как публичным так и приватным переменным

Деструктор - вызывается для удаления класса когда весь код отработал.Для освобождения памяти

Константу можем вызывать как в классе echo self::ConstantName так и вне класса echo ClassName::ConstantName.Имя константы пишем в верхнем регистре

        
            <?php   
            class User3{ 
                const PASS = 'password';
                const PA = 'password';

                function __construct($name,$surname,$login){  //**************************************************Функция конструктор
                    $this->name = $name;                      //Полученные параметры сразу кладет в поля объекта
                    $this->surname = $surname;
                    $this->login = $login;
        
                   $this->showAll3("Пользователь: " );  //вызывается сразу для всех созданных объектов
        
                   echo "<br>", self::PASS, "<br>";  //вызов константы внутри класса
                }
        
                function showAll3($text){

            }
