События мыши

        
            document.querySelector('.className').onclick = function (event) { 
                console.log(event); // Выводит объект события в консоль
            }
            
             onclick - клик
             ondblclick - двойной клик
             oncontextmenu - при нажатие на правую кнопку мыши вызывается контекст меню
                            если из футкции вернуть false то контекстное меню вызываться не будут
             onmausemove - навидение на элемент
             onmouseenter - при в заходе курсора на блок
             onmouseleave - при выходе курсора из блока
             onmousedown - когда навел курсор нажал и держиш
             onmouseup - когда отпуситил мыш
