Как пользоваться сайтом

Сайт организован в виде многоуровневой структуры:

  • Используйте навигационное меню слева для перехода между разделами
  • Каждый раздел содержит несколько глав
  • Вы можете скрыть меню кнопкой в верхнем правом углу навигации
  • Перетаскивайте границу между меню и контентом для изменения ширины
  • Используйте кнопку переключения темы для изменения цветовой схемы

Как добавлять контент

Для добавления нового контента:

  1. Создайте новый HTML-файл в папке chapters/sectionX/
  2. Добавьте ссылку на файл в navigation.html
  3. Для вставки кода используйте теги <pre><code class="language-xxx">
